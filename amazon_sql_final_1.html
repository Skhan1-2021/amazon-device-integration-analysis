Problem Statement

The team wants to label the usage of each device category into 'Low' or 'High' based on usage duration from July 1, 2024 to September 30, 2024. If the total usage time was less than 300 minutes, categorize it as 'Low'. Otherwise, categorize it as 'High'. Return a report with device ID, usage category, and total usage time.

SQL Solutions

Standard Solution
device_usage_basic.sql 
SELECT
    d.device_id,
    SPLIT_PART(d.device_name, ' ', 1) AS device_category,
    SUM(f.usage_duration_minutes) AS total_usage_time,
    CASE
        WHEN SUM(f.usage_duration_minutes) < 300 THEN 'Low'
        ELSE 'High'
    END AS usage_label
FROM fct_device_usage f
JOIN dim_device d ON f.device_id = d.device_id
WHERE f.usage_date BETWEEN '2024-07-01' AND '2024-09-30'
GROUP BY d.device_id, d.device_name;

Pro Solution (CTE)
device_usage_cte.sql 
WITH Device_Summary AS (
    -- Step 1: Clean device names and aggregate usage minutes
    SELECT
        d.device_id,
        SPLIT_PART(d.device_name, ' ', 1) AS device_category,
        SUM(f.usage_duration_minutes) AS total_usage_time
    FROM fct_device_usage f
    JOIN dim_device d ON f.device_id = d.device_id
    WHERE f.usage_date BETWEEN '2024-07-01' AND '2024-09-30'
    GROUP BY d.device_id, d.device_name
)
-- Step 2: Apply Low / High label using the pre-computed totals
SELECT
    device_id,
    device_category,
    total_usage_time,
    CASE
        WHEN total_usage_time < 300 THEN 'Low'
        ELSE 'High'
    END AS usage_label
FROM Device_Summary
ORDER BY total_usage_time DESC;
